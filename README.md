# ApertureF5
A bootstrap 5 based version of the DNN 10 Aperture Theme

## Intro
The DNN 10 Aperture Theme was created as a minimalist theme that is not supposed to be used in production.
I do not agree with this as IMO the default Theme should sell DNN.
So I decided to create a version which is usable as a full DNN Theme.
Also, I will use Bootstrap as a basis as that's what most DNN modules use.
