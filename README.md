# ApertureF5 DNN theme - Bootstrap 5

> A Bootstrap 5 based version of the DNN 10 Aperture theme — rebuilt for production use.

⚠️ **In development!**

---

### Intro

The DNN 10 Aperture theme was created as a minimalist theme that is not supposed to be used in production. I do not agree with this — in my opinion the default theme should sell DNN. So I decided to create a version which is usable as a full DNN theme. Bootstrap 5 is used as a basis, as that is what most DNN modules rely on.
